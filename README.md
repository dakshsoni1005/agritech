# 🌿 Gujarat AI Crop Recommendation & Smart Advisory Platform
### ગુજરાત એઆઈ પાક ભલામણ અને સ્માર્ટ કૃષિ સલાહકાર પ્લેટફોર્મ

An intelligent, data-driven crop recommendation engine and 18-field technical crop database built specifically for farmers across Gujarat's 5 regions: **North Gujarat, South Gujarat, Central Gujarat, Saurashtra, and Kachchh**.

---

## 🌟 Key Features

1. **🤖 AI Crop Recommendation Engine (એઆઈ પાક સલાહ)**:
   - Evaluates Region, District, Taluka, Soil Type, Sowing Month, Temperature, Rainfall, Irrigation availability, Farm Size (Hectare/Bigha), and Previous Crop history.
   - Outputs primary crop recommendation, confidence score (%), predicted yield range, farm-size adjusted net profit, AI reasoning, secondary options, and companion crop advice.

2. **📚 Comprehensive 18-Field Crop Database (સંપૂર્ણ પાક માહિતી કોષ)**:
   - Includes full technical profiles for Groundnut, Cotton, Cumin, Castor, Wheat, Mustard, Sugarcane, Kesar Mango, Pearl Millet, Potato, Fennel, Paddy, Chickpea, Date Palm, Sesame, etc.
   - Displays all 18 fields: Crop Name, Gujarati Name, Suitable Regions, Soil Types, Temp Range, Rainfall Range, Sowing Months, Harvest Months, Water Requirement, NPK Fertilizer Doses, Seed Rate, Average Yield, Diseases & Pests, Prevention Methods, Cultivation Cost, Market Price, Profit Range, and Rotation Crops.

3. **🗺️ 5 Agro-Climatic Regions Explorer (પ્રાદેશિક જમીન માર્ગદર્શિકા)**:
   - Interactive breakdown of North Gujarat, South Gujarat, Central Gujarat, Saurashtra, and Kachchh.

4. **🧮 Yield, NPK & Profit Estimator (નફો અને ખાતર ગણતરી)**:
   - Scalable calculator for farm size in Hectare or Bigha.

5. **🌐 Bilingual Support**:
   - One-click toggle between **Gujarati (ગુજરાતી)** and **English**.

---

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation & Execution

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
